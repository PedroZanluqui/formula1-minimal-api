# API Fórmula 1

API REST desenvolvida com Fastify que fornece informações sobre pilotos e equipes da Fórmula 1.

## Endpoints

- `GET /teams` - Lista todas as equipes
- `GET /pilots` - Lista todos os pilotos
- `GET /pilots/:id` - Busca piloto por ID

## Tecnologias

- Node.js
- TypeScript
- Fastify
